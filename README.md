## 🚀 Getting Started

### Prerequisites
```bash
# Python 3.8 or higher required
python --version

# Install Jupyter Notebook
pip install jupyter notebook
```

### Installation
```bash
# Clone the repository
git clone https://github.com/shubham-031/Gen_AI_Clg_Assignments.git

# Navigate to the repository
cd Gen_AI_Clg_Assignments

# Install required packages
pip install -r requirements.txt
```

### Running Assignments
```bash
# Launch Jupyter Notebook
jupyter notebook

# Open the desired assignment notebook and run cells
```
